# Packet Analysis Report

## DNS Analysis

Used the DNS filter to observe domain name resolution requests and responses generated while browsing websites.

## TCP Analysis

Analyzed TCP packets to understand how devices establish and maintain network connections for data transmission.

## TLS Analysis

Observed TLS traffic to understand how secure communication is established between a web browser and a website.

## Conclusion

This project provided hands-on experience with packet capture, protocol filtering, and basic network traffic analysis using Wireshark.
